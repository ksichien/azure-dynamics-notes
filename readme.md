# azure-dynamics-scripts
This is a small collection of PowerShell & SQL scripts I've written to automate the installation and configuration of Microsoft Dynamics NAV.

- configuration:
    - configure-aad-sync
    - configure-nav-auth
- database scripts:
    - create-db-user
    - get-db-role-members
    - rename-db
- setup:
    - create-nav-instance
    - create-nav-user
    - install-nav-client
